FTE

FTE(firstTimeExperience) API 는 유저의 최초 유입부터 자정까지의 행동만을 리포팅하는 "NewUserSession" 리포트를 위해 사용합니다.

아래 입력창에 샘플값을 입력하여 테스트합니다.

특수문자나 공백이 포함된 문자열은 반드시 URL ENCODING 처리하여 입력합니다.

RET

RET(retention) API 는 구매를 제외한 유저의 모든 행동을 리포팅하는 "In-App Activity" 리포트를 위해 사용합니다.

아래 입력창에 샘플값을 입력하여 테스트합니다.

특수문자나 공백이 포함된 문자열은 반드시 URL ENCODING 처리하여 입력합니다.

Purchase

Purchase API 는 유저 인앱 결제 행동을 리포팅하는 "Revenue Analytics" 리포트를 위해 사용합니다.

아래 입력창에 샘플값을 입력하여 테스트합니다.

OrderId
ProductId
ProductName
Price
Quantity
CurrencyCode
Category
